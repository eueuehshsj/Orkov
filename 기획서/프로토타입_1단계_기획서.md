# 데프 드레드 익스트랙션 슈터 - 1단계 프로토타입 기획서

## 전체 게임 컨셉 (참고용)
- 플랫폼: PC
- 장르: 탑다운 익스트랙션 슈터 (Escape from Duckov 스타일)
- 배경: 워해머 40K
- 주인공: 오크 데프 드레드
- 핵심 루프: 스테이지 입장 → 목표 달성 → 귀환(추출) → 자원 정산 → 강화/연구 → 다음 스테이지
- 리스크: 귀환 전 파괴 시 전 자원 손실(기본), 연구로 일부 보존/재생 등 완화 가능
- 자원: 종족별 자원(5등급) + 공용 자원(스크랩)
- 진영(MVP): 오크, 아스트라 밀리타룸, 타우, 티라니드 (스테이지별 단일 종족)
- 특화: 무기 = 해당 종족 공격방식 재현 / 방어구 = 해당 공격방식 카운터
- 신체 개조 = 장비 개조에 통합
- 이동속도 5단계 / 크기 4단계 (플레이어·적 공통)

---

## 1단계 프로토타입 범위

### 목적
핵심 전투+추출 루프(입장 → 전투 → 회수 → 귀환 → 강화)가 재미있는지 검증

### 대상 종족
아스트라 밀리타룸 (가드 보병)

### 스테이지 구성 (예시 스테이지)
- 단일 맵, 추출 포인트 1곳
- **메인 목표**: 특수 보병(엘리트) 처치 후 드랍 아이템 회수
- **보조 목표**: 모든 적 섬멸
- 메인 목표 클리어 후에도 반납(추출) 지점으로 이동해야 탈출 완료
  - → 보조 목표 수행 여부를 플레이어가 선택할 수 있는 구조 확인용

### 적 구성
- 일반 가드 보병 2종
  - 라이플병
  - 점사 사격병
- 특수 보병(엘리트) 1종
  - 일반 보병 기준 공격력 +30%, 체력 +30%
  - 패턴 차이 없음 (초기 단계는 스펙만 차별화)

### 플레이어(데프 드레드) 기본 스펙
- 이동속도: 보통
- 크기: 보통
- (가속/감속, 회전속도 등 "무거움" 표현은 1단계 미적용)
- 무기 슬롯: 1개

### 무기 2종
- **근접 클로**: 메인 데미지 딜러, 짧은 사거리, 높은 데미지
- **슬러거(원거리)**: 낮은 명중률, 근접 보조용 (메인보다 효율 낮음)

### 자원/강화
- 스크랩 1종
- 강화: "데미지 +", "방어 +" 등 단순 수치 강화 2~3단계

## 개발 순서 (1인 개발)
1. 전체적인 게임 구조 (씬 전환, 데이터 구조, 게임 루프)
2. 플레이어 이동
3. 무기 조작 (근접 클로 / 슬러거)
4. 적 AI (라이플병, 점사 사격병, 엘리트)
